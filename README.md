# MoviesExcel
This project focuses on cleaning and analyzing a movie dataset using Microsoft Excel. The goal of the project is to transform raw movie data into meaningful insights through data cleaning, lookup functions, and data visualization.

Tools Used
- Microsoft Excel
- Pivot Tables
- VLOOKUP
- Data Cleaning Techniques
- Data Visualization (Charts & Graphs)

Project Process
- Cleaned and structured the raw movie dataset to remove inconsistencies and missing values.
- Used VLOOKUP to combine and retrieve information across different data tables.
- Created Pivot Tables to summarize and analyze movie data efficiently.
- Built visualizations to highlight trends and patterns within the dataset.

Key Insights
The analysis helps explore patterns in movie data such as:
- Identifying which genres have the highest number of movies.
- Analyzing which genres tend to have the longest average movie duration.
- Determining which movies have the highest ratings and the overall average ratings.
- Examining which movies are the most popular based on the available data.
